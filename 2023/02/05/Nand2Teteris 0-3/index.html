Nand2Teteris 关于计算机组成

因为大学前两年专业学习的并不好，所以这个寒假打算从头夯实计算机基础，刚好在《CS自学指南》中看中了这门课，它的介绍语如下:

Coursera 上被数万人评为满分，在全球四百多所高校、高中被采用，让一个完全没有计算机基础的人从与非门开始造一台计算机，并在上面运行俄罗斯方块小游戏。

听起来就很酷对不对？实现起来更酷！这门课分为硬件和软件两个部分。在硬件部分，你将进入 01 的世界，用与非门构造出逻辑电路，并逐步搭建出一个 CPU 来运行一套课程作者定义的简易汇编代码。在软件部分，你将编写一个编译器，将作者开发的一个名为Jack的高级语言编译为可以运行在虚拟机上的字节码，然后进一步翻译为汇编代码。你还将开发一个简易的 OS，让你的计算机支持输入输出图形界面。至此，你可以用 Jack 开发一个俄罗斯方块的小游戏，将它编译为汇编代码，运行在你用与非门搭建出的 CPU 上，通过你开发的 OS 进行交互。学完这门课程，你将对整个计算机的体系结构有一个全局且深刻的理解，对于你后续课程的学习有着莫大的帮助。

你也许会担心课程会不会很难，但这门课面向的人群是完全没有计算机基础的人，课程作者的目标是让高中生都能理解。因此，只要你按部就班跟着课程规划走，一个月内学完应该绰绰有余。麻雀虽小但是五脏俱全，这门课很好地提取出了计算机的本质，而不过多地陷于现代计算机为了性能而设计出的众多复杂细节。让学习者能在轻松愉快的学习体验中感受计算机的优雅与神奇。

看完这个地方我挺激动的，完全从零开始搭建一个计算机，这也太棒了，但是同样的，我对这门课也有些自己的怀疑。（计算机组成原理这么难，从零开始搭建会不会根本不可能）但是还是抱着试一试的心态开始了这门课。

结果就是：听完Section 1和完成Project 1后，很快搭建了16位的与或非门，以及16位的选择器和解复用器(Dmux),这里其实还没有什么，很快在听完了Section 2之后我发现之前对我来说神秘不已的ALU的底层结构我居然可以很快地将它搭建出来（本质是多个16位的Mux的组合），我第一次发现原来底层的原理其实也并不怎么难，然后同样的很快的听完了Section 3之后我发现RAM我也可以实现了！！！！(时序电路的记忆功能先实现1Bit的读写然后到Register，RAM8，最后到RAM16k),我的老天爷，为什么不让我早点遇到这个课(当然我这一周就做到Projec 3,后面的课也不好说，到时候再回过头来做个总结)

Project 4-6(新的补充)

到目前算是基本完成了nand2tetris的hardware部分，这其中Project5和Project6都有一定难度，第五部分主要是让你完成一个CPU,第六部分则是写一个类似于汇编语言（但是比汇编语言简单）的编译器。CPU部分主要是要注意A instruction 与　C instruction 的不同情况（选择器真的是CPU一个很重要的部分）
编译器主要注意下有符号时需要预处理符号表的问题，总体来说也不算太难，只要肯花时间这门课一定会让你有一个特别nice的收获！！！

最后只想说,Nand2Teteris值得每一个计算机专业的学生去学一学，这门课相比于学校的计算机底层理论来说将会把你的底层知识弄得无比清晰与坚实
